The Elect is a one-person lab making small local voice models — fine-tuned 7B experiments built around distinct registers, personas, and sensibilities. This is the umbrella page for the fleet — Reformation prophets, register oracles, and the radical voices of history (most recently the women of revolution — Emma Goldman, Rosa Luxemburg, Louise Michel, Joan of Arc — and a Gnostic divine-voice oracle, Abraxas). Each model is a standalone artifact; all ship as Ollama-compatible GGUF files. The live roster: https://lerugray.github.io/the-elect/
